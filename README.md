# Homebrew Tap

Custom Homebrew tap for apps by [Julian Kahnert](https://github.com/JulianKahnert).

## Installation

```bash
brew tap juliankahnert/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `headsupkit` | macOS menu bar app that shows upcoming calendar events as a fullscreen overlay |

### Install

```bash
brew install --cask headsupkit
```
